Trong React, bạn có thể sử dụng state để điều khiển việc hiển thị/cập nhật các thành phần. Dưới đây là cách bạn có thể chuyển đổi mã này sang React:

jsx
import React, { useState } from 'react';

function Header() {
  const [isMenuActive, setIsMenuActive] = useState(false);

  const toggleMenu = () => {
    setIsMenuActive(!isMenuActive);
  };

  return (
    <div className={`header__menu ${isMenuActive ? 'active' : ''}`} onClick={toggleMenu}>
      {/* Content of your header menu */}
      <div className={`header__menu__items ${isMenuActive ? 'active' : ''}`}>
        {/* Your menu items */}
      </div>
    </div>
  );
}

export default Header;

Trong ví dụ này:

  • Chúng ta sử dụng hook useState để tạo một state isMenuActive, quản lý trạng thái của menu.
  • Hàm toggleMenu được sử dụng để chuyển đổi giá trị của isMenuActive khi menu được click.
  • Class "active" được thêm vào hoặc loại bỏ khỏi .header__menu và .header__menu__items dựa trên giá trị của isMenuActive.
  • Trong JSX, chúng ta sử dụng template literals để áp dụng class "active" dựa trên giá trị của isMenuActive.
